Version 3.0
Added
  • Modern text-only UI (no external images/backgrounds) for easy offline sharing.

  • Shared game navigation via nav.json.
  • Version badge in the title bar.
  • Header repeating in the Bag Items table (every 8 rows).

  • Rebuilt calculations to avoid duplicate HTML IDs and improve reliability.
  • Bag totals and grand totals displayed clearly at the bottom of the table.
Fixed
  • Fixed old calculation bug where totals could break due to undefined variables.
