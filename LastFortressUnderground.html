Notes

Tips for using this calculator offline.
  • Totals update as you type.
  • Headers repeat every 8 rows in the bag table for easier reading.
  • These calculators can be saved to your desktop or mobile device and opened locally in your browser.
  • If the top game menu doesn’t populate when opening as file://, run a tiny local server (example: python -m http.server).
